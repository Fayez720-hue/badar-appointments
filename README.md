#  نظام المعاينات والمتابعات -

تطبيق ويب لعرض وإدارة المعاينات (Appointments) والمتابعات (Callbacks) من Google Sheets.

## 🚀 كيفية الاستخدام

1. افتح التطبيق على الرابط: `https://YOUR-USERNAME.github.io/REPO-NAME/`
2. أدخل بيانات Google Sheets الخاصة بك:
   - **Spreadsheet ID**: من رابط جدول البيانات
   - **Sheet Name**: اسم الورقة (مثلاً Sheet1)
   - **API Key**: من Google Cloud Console
3. اضغط "حفظ واستخدام"

## 📋 هيكل الأعمدة المطلوب في Google Sheets

| العمود | الوصف |
|--------|-------|
| B | الاسم |
| C | الموبايل |
| O (15) | تاريخ المعاينة |
| P (16) | وقت المعاينة |
| L (12) | Callback Date (تاريخ المتابعة) |
| M (13) | Callback Notes |

## 🔧 إعداد Google Sheets API

1. اذهب إلى [Google Cloud Console](https://console.cloud.google.com/)
2. أنشئ مشروع جديد أو استخدم مشروعاً موجوداً
3. فعّل Google Sheets API
4. أنشئ مفتاح API (API Key)
5. شارك جدول البيانات الخاص بك مع "أي شخص لديه الرابط" (على الأقل للقراءة)

## 📱 الميزات

- عرض التقويم الشهري والأسبوعي
- إحصائيات فورية للمعاينات والمتابعات
- قائمة بالأحداث القادمة خلال 7 أيام
- حفظ الإعدادات محلياً في المتصفح
- دعم كامل للغة العربية
